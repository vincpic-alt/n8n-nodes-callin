# n8n-nodes-callin

This n8n community node enables seamless integration of **[Callin](https://callin.io) in your [n8n](https://n8n.io) workflows**.

**[Callin](https://callin.io)** offers AI-driven voice assistant platform that handles inbound and outbound customer conversations for businesses.

**[n8n](https://n8n.io)** is a powerful, AI-native, open-source workflow automation tool. Connect hundreds of services and rapidly create complex solutions with ease.

## Features

- Appointment Scheduling with upgraded Google Calendar Sync
- Lead Qualification & Call Transfers
- Reports, Recordings & Transcripts for analytics and training

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Credentials

To use this node, you need to have an [callin](https://callin.io) account and setup your api-key in the node built-in credentials.

## Compatibility

1.0.5
